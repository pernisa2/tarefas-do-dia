## 📝 Sobre o Projeto

O **Tarefas do Dia** é um gerenciador de listas dinâmico desenvolvido com HTML5, CSS3 e JavaScript puro. O foco principal deste projeto foi aplicar conceitos práticos de manipulação do DOM, criação dinâmica de elementos HTML via script, escuta de eventos de teclado/clique e estruturação de layouts limpos e responsivos.

## 📂 Código Fonte e Estrutura

Assim como meus outros projetos, o código é totalmente aberto para estudos e melhorias! A estrutura é composta por:
*   `index.html`: Estrutura base da aplicação (input de texto, botões e container da lista).
*   `script.js` (ou o arquivo correspondente de lógica): Responsável por capturar a entrada do usuário, renderizar novas tarefas na tela e gerenciar a remoção de itens.
*   Estilização dedicada para garantir uma experiência visual agradável e intuitiva para o usuário.

## 🕹️ Como Executar o Projeto

Para testar o projeto localmente na sua máquina:

1. Faça o download ou clone este repositório para o seu computador.
2. Navegue até a pasta do projeto e dê um duplo clique no arquivo `index.html`.
3. O aplicativo abrirá imediatamente no seu navegador padrão, pronto para uso!

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Criação dos campos de entrada de dados e marcação estrutural da lista.
* **CSS3:** Estilização moderna, alinhamento dos elementos e foco na usabilidade do usuário.
* **JavaScript (ES6):** Lógica funcional para adicionar novas tarefas, marcar itens concluídos e gerenciar a exclusão de registros.
